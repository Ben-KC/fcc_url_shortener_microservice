Free Code Camp "URL Shortener Microservice" Project

Example inputs:

  • thisapp.com/new/https://www.example.com
  • thisapp.com/new/http://example.com

Example outputs:

  • {"original_url": "https://www.example.com", "shortened_url": "thisapp.com/12345"}
  • {"original_url": "http://example.com", "shortened_url": "thisapp.com/45678"}

Example redirect:

  • thisapp.com/12345
  • will redirect to
  • https://www.example.com
